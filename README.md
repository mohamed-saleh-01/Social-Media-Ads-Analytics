# Social Media Advertising Analysis & Prediction

## 📌 Project Overview
This project presents an end-to-end data analysis and machine learning solution for evaluating social media advertising performance using a dataset from Kaggle.

It combines data cleaning, exploratory analysis, feature engineering, predictive modeling, and dashboard visualization to deliver actionable business insights.

---

## 📂 Dataset
- Source: Kaggle  
- Dataset: Social Media Advertising Dataset  
- Link: https://www.kaggle.com/datasets/jsonk11/social-media-advertising-dataset  

The dataset includes campaign data across:
- Facebook, Instagram, Twitter, Pinterest  

Key metrics:
- Impressions, Clicks, Engagement  
- Conversion Rate, Ad Spend  
- Campaign Goals, Demographics

## 🎯 Objectives
- Evaluate campaign performance across platforms  
- Understand audience behavior and segmentation  
- Measure ROI and marketing efficiency  
- Build a predictive model for campaign success  
- Deliver insights through an interactive dashboard  

---

## 🛠️ Tech Stack
- Python: Pandas, NumPy, Matplotlib, Seaborn  
- Machine Learning: Scikit-learn  
- Data Processing: Power Query  
- Visualization: Power BI  

---

## 🔄 Project Workflow
1. Data Cleaning: Handling missing values, duplicates, formatting  
2. EDA: Platform comparison, engagement trends, audience insights  
3. Feature Engineering: Engagement rate, ROI indicators  
4. Preprocessing: Encoding, scaling, train/test split  
5. Modeling: Gradient Boosting Regressor  

✅ Model Accuracy: 82%

---

## 📊 Dashboard Insights

The Power BI dashboard provides a comprehensive view of advertising performance:

- Audience Analysis:  
  Balanced engagement across genders with strong performance in key segments  

- Campaign Performance:  
  High competition between companies with consistent ROI (~3.18)  

- Platform Analysis:  
  Facebook & Instagram lead in CTR, while performance remains stable across all platforms  

📌 Key Insight:  
A multi-platform strategy ensures stable performance and consistent conversion rates (~8%)
